✨ معلومات سيرفر LB4 Legends ✨

🌍 الوصف ▶

LB4 Legends هو سيرفر Minecraft يجمع لاعبي Java و Bedrock في عالم واحد مليء بالمغامرات والتحديات. السيرفر خفيف، ممتع، ويعتمد على إضافات مميزة تجعل التجربة أكثر إثارة ومتعة.

⚔️ نظام الرتب ▶
  • 🛡️ Admin
  • 👑 Moderator
  • 📢 Chief guest
  • 🔥 Owner
💡 مميزات السيرفر ▶
  • ✅ يدعم Java من 1.8 حتى آخر إصدار.
  • 📱 يدعم Bedrock من الإصدار 1.21.100 وما فوق.
  • 🌟 إضافات مثل: ViaVersion, Geyser, OnePlayerSleep, TPA, WorldGuard, VeinMiner.
  • 🗺️ السيرفر غير دائم لكن عندك صلاحية تفتحو من Aternos متى تحب.
  • 🎖️ الرتب تُكتسب عبر مهمات يحددها الـOwner.
📜 القوانين ▶
  • 1️⃣ احترام جميع اللاعبين داخل السيرفر.
  • 2️⃣ يمنع السب أو الاستفزاز.
  • 3️⃣ يمنع استخدام أي هاك أو برامج غش.
  • 4️⃣ يمنع السرقة أو التدمير العشوائي (Griefing).
  • 5️⃣ لا تستعمل Bugs أو Glitches للاستفادة.
  • 6️⃣ استمتع وخلّي الجو مرح مع الجميع 😄
